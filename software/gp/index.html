Evolving Shapes
C++, OpenFrameworks, OpenGL
Genetic Programs are evolved to deform 3D shapes.
Parameters in the genetic program being manipulated in real time to show different manifestations of the resulting form.
Details and Implementation
Genetic Programs take a set of lines arranged in a 2D grid as input. They are then evolved to modify the position of these lines in 3D space. This is an exploration of the "character" of mathematical functions.
