Network Programs
JavaScript, D3.js, HTML, CSS
This is an experiment in which I modify genetic programs to have a network or graph structure that allows recursion. I call them "Network Programs".
Manipulating constraints in D3's force-directed network layout.
Details and Implementation
Genetic programs were originally implemented as trees where the computation enters at the root and flows out to the tip of each branch in turn, until the entire tree is traversed and the computation is complete.
In Network Programs a node can have any other node as a child, allowing the computation to loop back around and reenter itself at any point.
Infinite loops become possible and are stopped by a computation maximum depth limit. Once the computation has descended through the maximum allowed number of children, the final node, if it is not a terminal itself, will be evaluated using an attached "leaf" node which is always a terminal with no children. Each node in the network has its own personal leaf node for cases where it becomes the final node in a max depth evaluation.
Black nodes display their function, e.g. "pow" where the first child is raised to the power of the second child, or "neg" which negates the value of its only child. The single green node is the root, where the computation initially enters the network. Leaf nodes are shown as smaller red circles attached to each node.
Sliders are connected to some of the constraints in D3's force directed graph layout to test different values.

The output of the Network Program is displayed in a graph. The Network Program is like a genotype, and the graph is the resulting animal or phenotype. Often the result is an interesting and unusual curve that appears to have a particular aesthetic style, like a personality.
