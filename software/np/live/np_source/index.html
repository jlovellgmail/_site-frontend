"Network Programs" Visualization
JavaScript, D3.js, HTML, CSS
This is an experiment where I'm modifying genetic programs to have a network or graph structure that allows recursion. I call them "Network Programs".
Details and Implementation
Our forefathers created genetic programs with a unidirectional tree structure where the computation enters at the root and flows out to the tip of each branch in turn, until the entire tree is traversed and the computation is complete.
In Network Programs nodes can have any other node as a child, so they can loop back around and reenter the computation at any point.
Infinite loops become possible in this structure and are stopped by a computation maximum depth limit. Once the computation has descended through the maximum allowed number of children, the final node, if it is not a terminal itself, will be evaluated using an attached "leaf" node which is always a terminal with no children. Each node in the network has its own personal leaf node for cases where it becomes the final node in a max depth evaluation.
Black nodes display their function, e.g. "pow" where the first child is raised to the power of the second child, or "neg" which negates the value of its only child. The single green node is the root, where the computation initially enters the network. Leaf nodes are shown as smaller red circles attached to each node.
Sliders are connected to some of the constraints in D3's force directed graph layout to test different values.
I display the output of the Network Program in a graph. The Network Program is like a genotype, and the graph is the resulting animal or phenotype. A lot of the time nothing is visibiel in the graph because, for instance, all the output went to zero or infinity. But often the result is an interesting and unusual curve that appears to have a particular aesthetic style, like a personality.
